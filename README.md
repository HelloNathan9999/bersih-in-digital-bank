## ğŸ“‹ Instalasi & Development

### Prerequisites
- Node.js 18+ dan npm
- Git

### Setup Development
```bash
# Clone repository
git clone <YOUR_GIT_URL>
cd bersih-in

# Install dependencies
npm install

# Start development server
npm run dev
```

### Mobile Development
```bash
# Add platforms
npx cap add ios
npx cap add android

# Build and sync
npm run build
npx cap sync

# Run on device/emulator
npx cap run android
npx cap run ios  # Requires macOS + Xcode
```
